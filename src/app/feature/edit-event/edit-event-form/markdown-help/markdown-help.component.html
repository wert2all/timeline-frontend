Headings

Use the hash symbol (#) to create headings of different levels:

  • # Heading 1
  • ## Heading 2
  • ### Heading 3
  • ...

Emphasis

Use asterisks (*) or underscores (_) for emphasis:

  • *italic* or _italic_
  • **bold** or __bold__
  • ***bold italic***

Lists

Create unordered lists with hyphens (-) or asterisks (*):

    - Item 1
    - Item 2

Create ordered lists with numbers followed by periods:

    1. Item 1
    2. Item 2

Code

Use backticks (`) for inline code:

For code blocks, use triple backticks (```):

    ```
    code block
    ```

Tables

Create tables using pipes (|) and hyphens (-):

    | Header 1 | Header 2 |
    |---|---|
    | Row 1, Cell 1 | Row 1, Cell 2 |
    | Row 2, Cell 1 | Row 2, Cell 2 |

Horizontal Rules

Create horizontal rules with three or more hyphens (---) or asterisks (***): 

    ---
